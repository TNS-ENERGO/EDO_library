Единицы измерения

Справочник предназначен для регистрации перечня единиц измерений, определенных для позиций номенклатуры.

Список единиц измерения можно заполнить автоматически на основании общероссийского классификатора единиц измерения (ОКЕИ) по кнопке Подобрать из классификатора. В открывшейся форме нужно выбрать единицу измерения и добавить ее в список по кнопке ОК.
