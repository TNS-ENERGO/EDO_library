Номенклатура

Справочник предназначен для хранения информации о номенклатурных позициях следующих типов: товар, услуга, работа и право.

В данный справочник заносится следующая информация:

  • Вид номенклатуры - дополнительная классификация, предназначенная для объединения номенклатуры по общим признакам учета.
  • Рабочее наименование - наименование, используемое для отражения в экранных формах в программе.
  • Полное наименование - наименование, используемое при формировании электронных документов.
  • Входит в группу - группа справочника, в которую включен элемент. При изменении реквизита происходит перемещение текущего элемента из старой группы в новую.
  • Артикул - цифровой код товара.
  • Единица измерения - каждая номенклатурная позиция характеризуется учетной единицей измерения из справочника Единицы измерения.
  • Ставка НДС - используется в качестве значения по умолчанию в документах поступления и реализации.
  • Комментарий - Произвольное текстовое описание номенклатурной позиции.
