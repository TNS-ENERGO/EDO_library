Карточка

В форме отображается карточка торгового предложения. Выводится информация о товаре, его характеристиках, поставщике, контактные данные поставщика.

Из формы просмотра отображаются гиперссылки для открытия профиля поставщика и формы поиска товаров по категории товара.

По команде Добавить в корзину товар добавляется в корзину формы поиска, из которой открывается карточка товара.
